# Lakshya Portfolio

Single-page portfolio built with Vite, React, TypeScript, and Tailwind CSS — inspired by a modern designer/developer hero layout.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build for production

```bash
npm run build
npm run preview
```

## Customize content

Edit [`src/data/site.ts`](src/data/site.ts) for name, bio, email, social links, projects, and services.

## Hero portrait

Your photo lives at [`public/portrait.png`](public/portrait.png). Replace that file to update the hero and about sections — no code changes needed.

Recommended size: ~800×1000px PNG with a clean or transparent background.

## Deploy

Static output is in `dist/` after `npm run build`. Deploy to Vercel, Netlify, or GitHub Pages.
